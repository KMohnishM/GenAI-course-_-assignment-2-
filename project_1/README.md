# Mini Project 1: Data Science with Generative AI

This project contains a complete, automated end-to-end data science pipeline using the **Student Performance** dataset from the UCI Machine Learning Repository. It has been set up with both local verification scripts and a Google Colab-ready notebook (`notebook.ipynb`) to fulfill the assignment requirements.

---

## How to Run

### Option 1: Using Google Colab (Recommended)
1. Upload the `notebook.ipynb` file to your [Google Colab](https://colab.research.google.com/).
2. Run the cells in order. The notebook will automatically download the dataset, perform ETL, database setup, EDA, and model training, and output everything.
3. You can download the generated plots from the file explorer on the left sidebar in Colab.

### Option 2: Running Locally
1. Install dependencies:
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn scipy shap
   ```
2. Run the main pipeline script:
   ```bash
   python run_pipeline.py
   ```
   This will:
   - Clean the raw data and save it to `gold/clean_data.csv`.
   - Setup the SQLite database and insert records.
   - Run the EDA analysis, hypothesis tests, and train all machine learning models.
   - Export all plots directly to the `outputs/` folder.
   - Generate `notebook.ipynb` (Jupyter notebook) and `submission_draft.md` (report text content).
